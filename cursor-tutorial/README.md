# Cursor入門 - 学習ガイド

## この章の目的

- Cursor（AI搭載テキストエディタ）を使い始める前に、**テキストエディタの本質**と**プロジェクトの概念**を理解する。
- **Markdown（mdファイル）**の書き方を学び、仕様書やレポートをテキストベースで作成できるようになる。
- Word・Googleドキュメントなど普段使うツールとの違いを明確にし、適切な場面で使い分けられるようになる。
- Cursor特有の概念（モデル・エディタ・作業空間・コンテキスト）を学び、効果的に活用するための土台を作る。

## 対象者

- Cursorをこれから使い始める非エンジニア（Webディレクター、マーケティング担当者想定）
- テキストエディタやコード編集環境に馴染みのない方
- プログラミング経験はないが、コードを読みながら学習したい方
- 仕様検討やマーケティングリサーチなどでCursorを活用したい方

## 前提・推奨環境

- OS: Windows 11（本教材は Windows 前提）
- Cursor がインストール済みであること（またはこれからインストールする予定）
- ネットワーク接続（AI機能利用時）

## この章で身につく力

- テキストエディタと文書作成ソフトの違いを自分の言葉で説明できる
- 作業内容に応じて適切なツールを選べる
- 「プロジェクト」という言葉を開発者の文脈で理解できる
- Cursorのモデルとエディタの役割を区別して活用できる
- 作業空間とコンテキストを意識した効率的な作業ができる

## 学習のコツ

- **比喩で理解する**：テキストエディタは「原稿用紙」、Wordは「完成した本のデザイン」のように、イメージで捉える。
- **実物を触る**：説明を読んだら、実際にCursorでフォルダを開いて確認する。
- **違いを意識する**：同じファイルをWordとCursorの両方で開き、表示の違いを比較してみる。
